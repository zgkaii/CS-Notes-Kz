# 结构层次

## 逻辑结构

MongoDB 的逻辑结构是一种层次结构。主要由：文档(document)、集合(collection)、数据库(database)这三部分组成。

<img src="..\..\..\images\MongoDB\逻辑结构.png" alt="img" style="zoom: 80%;" />

与关系型数据库对比：

| MongoDB          | 关系型数据库     |
| ---------------- | ---------------- |
| 数据库(database) | 数据库(database) |
| 集合(collection) | 表(table)        |
| 文档(document)   | 行(row)          |
